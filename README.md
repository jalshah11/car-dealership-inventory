# Car Dealership Inventory System

A full-stack car dealership inventory system, built incrementally with
Test-Driven Development.

- `backend/` — Node.js + TypeScript + Express REST API, PostgreSQL via Prisma
- `frontend/` — React + TypeScript + Vite SPA (coming in a later session)

This README will grow into full project documentation (architecture, API
reference, database schema, setup instructions, test report, and an "AI
Usage" section) as the project progresses. See `backend/prisma/README.md`
for current Prisma/database setup notes.
